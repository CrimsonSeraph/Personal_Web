# CrimsonSeraph 的个人网站

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE.txt)
[![Hugo Version](https://img.shields.io/badge/Hugo-v0.121.2+-blue)](https://gohugo.io/)

这里是 [CrimsonSeraph](https://github.com/CrimsonSeraph) 的个人网站。本站基于 [Hugo](https://gohugo.io/) 静态站点生成器构建，使用主题 [D-Sketon/hugo-theme-reimu](https://github.com/D-Sketon/hugo-theme-reimu) 用于记录个人生活、学习与项目展示等内容。

网站已通过 Cloudflare Pages 自动部署，可通过以下地址访问：
🔗 **[https://personal-web.crimsonseraph.top/](https://personal-web.crimsonseraph.top/)**

## ✨ 主要特性

*   **主题风格**：基于优雅的 [Reimu](https://github.com/D-Sketon/hugo-theme-reimu) 主题进行定制和优化，提供了舒适的阅读体验。
*   **评论互动**：集成了 [Waline](https://waline.js.org/) 评论系统，支持访客在文章下方进行留言交流。
*   **全文搜索**：通过 [Algolia](https://www.algolia.com/) 提供快速、可靠的站内全文搜索功能。
*   **访问统计**：使用 [moe-counter-cf](https://github.com/SunDoge/moe-counter-cf) 在 Cloudflare Workers 上部署了网站计数器，用于统计总访问量与访客数。
*   **原创 Logo**：网站 Logo (`favicon.ico`) 为个人原创设计，**未经许可，禁止转载或盗用**。

## 🛠️ 技术栈与依赖

*   **核心框架**：[Hugo](https://gohugo.io/) - 极快的 Go 语言静态网站生成器。
*   **网站主题**：[D-Sketon/hugo-theme-reimu](https://github.com/D-Sketon/hugo-theme-reimu) - 为本项目提供基础样式与布局。
*   **评论系统**：[Waline](https://waline.js.org/) - 一款简洁、安全的评论系统，后端基于 LeanCloud。
*   **站内搜索**：[Algolia](https://www.algolia.com/) - 提供强大的实时搜索 API。
*   **访问计数器**：[SunDoge/moe-counter-cf](https://github.com/SunDoge/moe-counter-cf) - 部署在 Cloudflare Workers 上的可爱风格计数器。
*   **部署与托管**：[Cloudflare Pages](https://pages.cloudflare.com/) - 连接 GitHub 仓库，实现自动化构建与全球分发。
