# CrimsonSeraph's Personal Website

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE.txt)
[![Hugo Version](https://img.shields.io/badge/Hugo-v0.121.2+-blue)](https://gohugo.io/)

This is the personal website repository of [CrimsonSeraph](https://github.com/CrimsonSeraph). Built with the [Hugo](https://gohugo.io/) static site generator and the [D-Sketon/hugo-theme-reimu](https://github.com/D-Sketon/hugo-theme-reimu) theme, it serves as a space to document personal life, learning notes, and project showcases.

The site is automatically deployed via Cloudflare Pages and can be accessed at:
🔗 **[https://personal-web.crimsonseraph.top/](https://personal-web.crimsonseraph.top/)**

## ✨ Key Features

*   **Theme Style**: Customized and optimized based on the elegant [Reimu](https://github.com/D-Sketon/hugo-theme-reimu) theme, providing a comfortable reading experience.
*   **Comments & Interaction**: Integrated with the [Waline](https://waline.js.org/) comment system, allowing visitors to leave messages and interact below articles.
*   **Full-Text Search**: Powered by [Algolia](https://www.algolia.com/) for fast and reliable on-site full-text search.
*   **Visit Statistics**: Uses [moe-counter-cf](https://github.com/SunDoge/moe-counter-cf) deployed on Cloudflare Workers to count total page views and unique visitors.
*   **Original Logo**: The website logo (`favicon.ico`) is an original design by the author. **Unauthorized reproduction or misappropriation is prohibited.**

## 🛠️ Tech Stack & Dependencies

*   **Core Framework**: [Hugo](https://gohugo.io/) - A fast Go-based static site generator.
*   **Theme**: [D-Sketon/hugo-theme-reimu](https://github.com/D-Sketon/hugo-theme-reimu) - Provides the base styles and layout.
*   **Comment System**: [Waline](https://waline.js.org/) - A simple, secure comment system with a LeanCloud backend.
*   **Search Engine**: [Algolia](https://www.algolia.com/) - Offers powerful real-time search APIs.
*   **Visit Counter**: [SunDoge/moe-counter-cf](https://github.com/SunDoge/moe-counter-cf) - A cute-style counter deployed on Cloudflare Workers.
*   **Deployment & Hosting**: [Cloudflare Pages](https://pages.cloudflare.com/) - Connects to the GitHub repository for automated builds and global distribution.
